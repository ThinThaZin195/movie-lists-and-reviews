# Movie Lists and Reviews

A movie browsing web application where users can explore movie lists and reviews.

## Live Demo

https://movie-lists-and-reviews-5ll8.vercel.app/

## Features

- Browse movie lists
- View movie posters and information
- Responsive UI
- Fetch movie data from a public API

## Tech Stack

- Next.js
- React
- JavaScript
- CSS
- Movie API

## Installation

Clone the repository

git clone https://github.com/ThinThaZin195/movie-lists-and-reviews

Install dependencies

npm install

Run the development server

npm run dev

Open http://localhost:3000 in your browser.

## Deployment

This project is deployed using Vercel.
